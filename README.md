# j-studio

not a template. not a fork. built from scratch with next.js, typescript, and an unreasonable amount of vanilla css.

the theme toggle is a pull-chain. like a lamp. because i wanted one.

no tailwind. no component libraries. no fancy animations. just blueprint grids, clean borders, and lowercase everything.

if you're here to copy-paste, the content lives in `/data`. change it. don't ship my name as yours.

`npm i && npm run dev` — you'll figure out the rest.
